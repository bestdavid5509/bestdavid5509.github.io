# Portfolio Website — Conventions

## Git
- No "Co-Authored-By: Claude" attribution in commit messages
- Conventional commit format (feat:, fix:, docs:, chore:, etc.)
- Concise commit messages — subject line + brief body when useful

## Workflow
- Survey current file state before edits
- PAUSE before pushing for interactive verification in browser preview
- Commit after each logical change (don't batch unrelated edits)
